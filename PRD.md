# 数据报表后台 — 产品功能需求方案

> 基于 cetus-pm demo 整理，版本 v1.0，2026-03-29

---

## 一、产品概述

### 1.1 产品定位

面向广告投放运营团队的**数据报表后台**，提供多维度数据的查询、筛选、聚合分析能力，支持视图保存分享和快捷标签管理，帮助运营人员高效完成日常数据复盘与分析。

### 1.2 核心用户

- 广告投放运营（主要用户）
- 数据分析师
- 运营管理者（查看/共享视图）

### 1.3 核心价值

| 价值点 | 说明 |
|--------|------|
| 多维度灵活筛选 | 支持时间、游戏、渠道、优化师等多维组合筛选 |
| 快捷标签体系 | 将常用渠道组合沉淀为标签，一键切换分析视角 |
| 视图保存与分享 | 保存分析配置，支持团队内按权限共享 |
| 多 sheet 分析 | 同一页面支持多个分析维度并行，按粒度区分 |
| 指标过滤 | 在数据结果层按指标条件二次筛选 |

## 二、功能模块详述

### 2.1 顶部导航（TopNav）

**功能描述：**
- 展示产品 logo 和名称
- 提供跨平台主导航菜单
- 快捷入口：帮助文档、消息通知、当前环境标识、用户信息

**交互规则：**
- 用户信息点击展开用户菜单（登出、个人设置等）
- 环境标识区分测试/生产环境

---

### 2.2 侧边导航（Sidebar）

**功能描述：**
- 展示广告平台层级菜单（平台 → 产品线/账户组）
- 支持展开/收起子菜单
- 点击叶节点切换当前分析的平台/账户范围

**交互规则：**
- 当前选中项高亮
- 默认展开第一个平台

---

### 2.3 视图管理

#### 2.3.1 视图概念

**视图**是一份保存的分析配置快照，包含：
- 筛选条件（日期范围 + 各维度筛选值）
- 激活的快捷标签
- 时间粒度设置
- 聚合维度配置

#### 2.3.2 视图选择（ViewBar）

**功能描述：**
- 下拉选择已保存视图，切换后自动加载对应配置
- 置顶视图以标签形式常驻展示，支持单击快速切换
- 选中视图后自动重置至第一个 sheet

**置顶规则：**
- 最多置顶 10 个视图
- 置顶视图按置顶顺序排列

#### 2.3.3 保存视图（SaveMenu）

提供三种保存操作：

| 操作 | 说明 |
|------|------|
| 更新当前视图 | 将当前配置覆盖到已选视图（需先选中视图） |
| 另存为新视图 | 以当前配置创建新视图，需填写名称 |
| 缓存上次操作 | 退出后保留当前未保存的筛选状态 |

**另存为新视图** 支持设置：
- 视图名称（最多 30 字符，中文按 2 个字符计算）
- 时间类型：相对时间（今天/昨天/本周等）或绝对时间范围

#### 2.3.4 视图分享（ShareViewModal）

**分享模式：**

| 模式 | 说明 |
|------|------|
| 私有 | 仅自己可见 |
| 指定成员 | 选择组织架构中的特定成员，被选成员可查看 |
| 公开 | 所有人可见 |

**权限对齐机制：**

若视图包含快捷标签，且被分享用户对某标签无访问权限，系统弹出提示：
- 提示受影响的用户和标签
- 可选择：同步扩大标签权限 / 仅分享视图（用户看不到该标签）

#### 2.3.5 权限访问控制

当用户访问包含自己无权查看标签的视图时：
- 展示无权限空态页面
- 显示受限标签列表及标签所有者信息
- 不加载任何数据

---

### 2.4 筛选体系

#### 2.4.1 日期范围筛选（FilterBar 固定项）

**功能描述：**
- 始终展示，不可隐藏
- 支持手动输入日期或通过日历选择
- 日历支持快捷选项：今天、昨天、近 7 天、近 30 天、本周、本月、上周、上月

**交互规则：**
- 默认展示当月范围
- 日期变更后不自动触发查询，需手动点击"查询"按钮

#### 2.4.2 维度筛选 chips

**可筛选维度：**

| 维度 | 输入方式 | 说明 |
|------|---------|------|
| 消耗时间 | 日期范围选择器 | 固定项，不可移除 |
| 游戏名称 | 多选下拉 | 支持搜索 |
| 优化师 | 多选下拉 | 支持搜索 |
| 主渠道 | 多选下拉 | 受快捷标签锁定影响 |
| 子渠道 | 多选下拉 | 受快捷标签锁定影响 |
| 广告创建时间 | 日期范围选择器 | 可选项 |
| 账户 ID/名称 | 文本输入 | 支持批量粘贴 |
| 广告 ID/名称 | 文本输入 | 支持批量粘贴 |
| 计划 ID/名称 | 文本输入 | 支持批量粘贴 |
| 消耗区间 | 数值范围 | 含价格/ROI |
| 地区 | 多选下拉 | |
| 素材类型 | 多选下拉 | |

**通用筛选交互规则：**
- 多选 chip：支持列表模式（搜索+勾选）和批量输入模式（粘贴多行文本）
- 精确/模糊匹配：可切换，影响接口查询方式
- 排除模式：勾选后取反，筛选"不含这些值"的数据
- hover 时展示 × 快速清空按钮

**渠道锁定规则：**
- 当有快捷标签处于激活状态时，主渠道和子渠道筛选自动锁定
- 锁定状态下点击渠道筛选，弹出 toast 提示
- 原因：快捷标签已包含渠道维度，避免冲突

#### 2.4.3 全量筛选项选择（AllFiltersPopover）

**功能描述：**
- 以分组网格形式展示所有可选筛选维度
- 勾选后对应维度的 chip 出现在筛选栏
- 取消勾选后该维度的筛选值同时清空

**筛选分组：**
- 媒体相关：媒体、主渠道、子渠道、账户ID/名称
- 广告相关：游戏名称、优化师、地区、素材类型
- 时间相关：广告创建时间
- 数值相关：消耗区间

---

### 2.5 快捷标签

#### 2.5.1 标签概念

快捷标签将**特定渠道组合**封装为一个有颜色标识的标签，便于快速切换渠道视角进行分析。

#### 2.5.2 标签激活

- 点击标签 chip 切换激活/未激活状态
- 激活时：该标签关联的渠道自动应用为渠道筛选条件
- 激活时：主渠道/子渠道筛选锁定，不可手动修改
- 支持同时激活多个标签（渠道取并集）

#### 2.5.3 标签管理（QuickTagModal）

**标签属性：**

| 属性 | 说明 |
|------|------|
| 标签名称 | 最多 20 字符 |
| 颜色 | 8 种预设颜色 + 自定义 hex 颜色 |
| 主渠道 | 3 级级联选择（游戏 → 媒体 → 渠道） |
| 子渠道 | 自定义文本标识符 |
| 可见性 | 私有 / 指定成员 / 公开 |
| 授权成员 | 可见性为"指定成员"时，选择可查看的用户 |

**操作权限：**
- 自己创建的标签：可编辑、删除
- 他人创建的公开标签：仅可查看
- 被分享标签：可查看，不可编辑

**删除限制：**
- 若标签被某个共享视图使用，删除时系统提示，需先从视图中移除标签才可删除

#### 2.5.4 标签排序

- 支持拖拽标签 chip 调整显示顺序
- 排序保存至用户个人配置

---

### 2.6 多 Sheet 分析

#### 2.6.1 Sheet 概念

每个 sheet 代表一个独立的分析视角，拥有独立的：
- 时间粒度（天/周/月）
- 聚合维度配置
- 指标筛选配置
- 维度自动更新开关

#### 2.6.2 Sheet 管理（SheetTabBar）

| 操作 | 说明 |
|------|------|
| 新建 | 点击 + 按钮，创建默认 sheet |
| 重命名 | 右键菜单 → 重命名，保留时间粒度前缀 |
| 复制 | 右键菜单 → 复制，复制当前 sheet 的所有配置 |
| 删除 | 右键菜单 → 删除，至少保留 1 个 sheet |
| 拖拽排序 | 拖拽 sheet 标签调整顺序 |

**Sheet 命名规则：**
- 格式：`{粒度前缀}-{自定义名称}`，如 `天-投放分析`、`周-ROI汇总`
- 切换时间粒度时，前缀自动更新
- 粒度前缀：天 / 周 / 月

#### 2.6.3 时间粒度（TableToolBar）

三种粒度：天、周、月，对应不同的数据聚合周期。

切换粒度时：
- sheet 名称前缀同步更新
- 表格数据重新聚合

---

### 2.7 表格数据与聚合

#### 2.7.1 聚合维度（AggregateDimensionPopover）

**功能描述：**
- 选择按哪些维度对数据进行聚合分组
- 时间维度始终存在，不可取消
- 其他维度：游戏、媒体、主渠道、子渠道、优化师、地区等

**维度排序：**
- 默认排序：固定系统预设顺序
- 自定义排序：拖拽调整维度列顺序

**自动更新 sheet 名称：**
- 开启后，每次修改聚合维度，sheet 名称中的维度信息自动同步更新
- 此配置按 sheet 独立保存

#### 2.7.2 视图模式（TableToolBar）

| 模式 | 说明 |
|------|------|
| 普通视图 | 每条数据独立显示为一行 |
| 聚合视图 | 相同维度值的数据合并行展示（合并单元格） |

#### 2.7.3 指标筛选（MetricFilterPopover）

**功能描述：**
- 在查询结果上按指标条件二次过滤（本地过滤，不重新请求）
- 支持保存为命名筛选组合，供后续复用

**筛选组合结构：**
- 一个组合由多个"条件组"组成（组间为 OR 逻辑）
- 每个条件组包含多个条件（条件间为 AND 逻辑）
- 条件格式：`指标 操作符 值`（如：消耗 > 1000）

**操作符：** 大于、小于、大于等于、小于等于、等于、不等于

#### 2.7.4 局部筛选（LocalFilterPopover）

**功能描述：**
- 针对表格各维度列进行本地快速过滤
- 支持多值勾选、文本搜索、排除模式
- 不触发后端查询

**支持的筛选类型：**
- 列表选择型（游戏、媒体等）
- 文本输入型（账户ID、广告ID等，支持批量粘贴）
- 日期范围型（广告创建时间）

---

### 2.8 数据表格（DataTable）

#### 2.8.1 列结构

- **维度列**（左侧固定）：根据聚合维度配置动态生成
- **指标列**（右侧滚动）：展示各项投放指标数据

#### 2.8.2 合并单元格

聚合视图下，同一维度值的连续行自动合并，提升可读性。

#### 2.8.3 指标过滤高亮

当指标筛选激活时，不满足条件的行降低展示优先级（视觉弱化或隐藏）。

---

### 2.9 导出（ExportModal）

**功能描述：**
- 选择要导出的 sheet（可全选或单选）
- 导出包含当前筛选条件下的所有数据
- 格式：Excel（.xlsx）

## 三、关键业务规则

### 3.1 快捷标签权限体系

```
标签可见性：
  私有   → 仅创建者可见
  指定成员 → 创建者 + 被授权成员可见
  公开   → 所有人可见

视图分享时的权限对齐：
  1. 检查视图关联的所有标签
  2. 找出被分享用户无权访问的标签
  3. 提示运营：可选择扩大标签权限 或 保持不变（用户看不到该标签）
```

### 3.2 渠道锁定规则

```
当 quickTags.some(t => t.active) === true 时：
  主渠道筛选 → 锁定，不可操作
  子渠道筛选 → 锁定，不可操作
  点击锁定项 → toast 提示"快捷标签选中时，主/子渠道筛选不可用"
```

### 3.3 Sheet 命名规则

```
sheet 名称 = 粒度前缀 + "-" + 自定义部分
粒度前缀：天 | 周 | 月

切换粒度时：
  旧名称 "天-投放分析" → 新名称 "周-投放分析"

自动更新维度名称（开启时）：
  维度变更后，sheet 名称中追加维度标识
  如：天-媒体-优化师
```

### 3.4 置顶视图上限

```
pinnedViews.length >= 10 时，置顶按钮禁用
用户点击时 toast 提示
```
